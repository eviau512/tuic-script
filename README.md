# tuic script

An one-click Tuic deploy script

## Deploy

### tuic V4 + V5

```shell
wget -N --no-check-certificate https://gitlab.com/Misaka-blog/tuic-script/-/raw/main/tuic.sh && bash tuic.sh
```
